## Install
[![NPM](https://nodei.co/npm/labelx-cli.png?compact=true)](https://nodei.co/npm/labelx-cli/)
> `npm i labelx-cli -g`

## Usage

```bash
$ bax -h


  Usage: bax <command>

  Options:

    -v, --version  output the version number
    -h, --help     output usage information

  Commands:

    init           init LabelX Dev

...
```

## Init LabelX dev

```bash
$ bax init [name]
```
